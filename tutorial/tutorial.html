Tutorial

Welcome to the GymQR tutorial!

Please note that both the GymQR app and this tutorial are currently under development.

At this stage, the tutorial may not cover all the functionalities of the app.

The primary purpose of this tutorial is to visually guide you through the application using images.

If you encounter any issues with the display, such as elements being invisible, inaccessible, or not fitting the screen properly, don't hesitate to report these to the developer.

Beneath each image, you'll find a brief description explaining the purpose of each screen and the functions of the buttons.

If you have any questions about using the app, find certain functionalities challenging, or have suggestions for improvement, please reach out and let me know.

Your time spent exploring the app, testing its functions, and providing feedback is highly appreciated. Feel free to report any problems or suggest possible improvements.

GymQR is a web application that allows gym owners to manage their gym memberships and track attendance. It is able to generate unique QR codes for each member, which can be sent to the member via email. The QR code can be scanned by the GymQR app, which is used by gym members to check in and out of the gym. The app also can send predetermined reminder emails if the member has a payment due.
